# Smart-door-control
## Подробности технологического характера
1) Языки программирования: Java, Python
2) Инструменты разработки: Android Studio, VS Code  
3) Протокол взаимодействия HTTP/REST API  
4) Формат данных: JSON  
5) Микрокомпьютер: Raspberry Pi  
## План работ
1) Подготовка Raspberry Pi
2) Создание и запуск сервера на Python Fast API  
3) Создание Android-клиента на Java 
4) Тестирование и отладка
